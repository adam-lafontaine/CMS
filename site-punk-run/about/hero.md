## About this game

Punk Run is a cyberpunk themed game made with pixel-art assets from <https://itch.io/c/2053292/cyberpunk-pixel-art>

### Why the name 'Punk Run'?
The game is named after the animation spritesheet for the main character (Punk_run.png).

![Punk_run.png](https://github.com/adam-lafontaine/CMS/raw/sm-current/site-punk-run/img/Punk_run.png)

### Why use free pixel-art?
Because I am a programmer.  Making art is well outside of my wheelhouse.  Also, 'free' is within budget.

### What is the game about?
So far just running, jumping and spawing random sprites on screen.  The rule for development is that only art from the above asset packs can be used for the game.  Anything else is on the table

### What's next?
TBD.  Something new is added with each update.

### What game engine is used?
None.  Just old fashioned C++ with SDL2(3).  The game will be heavy on procedural generation so, I want full control of the game logic.  SDL allows for targeting multiple platforms without having to deal with the idiosyncrasies of operating system and graphics APIs.
